Animaciones

Una animación es la ilusión de movimiento que se crea al mostar una serie de imagénes de manera rápida y constante.

Propiedades de una animación

  • Duración
  • Periodicidad
  • Frecuencia de actualización
  • Interpolación

¿Para qué?

  • Añadir interés visual
  • Generar interactividad
  • Mostrar relaciones y jerarquías
  • Transmitir estados e ideas

Tipos de animaciones

  • Animaciones (
    animation
    , 
    @keyframes
    ):
    • Pueden correr una o más veces.
    • Empiezan automaticamente por lo general.
    • Mayor control, más trabajo crearlas.
  • Transiciones (
    transition
    ):
    • Corren solo una vez.
    • Empiezan mediante eventos o cambios de estado.
    • Mejor interacción con JavaScript.

Animaciones

Creacion de keyframes

            
/*regla @keyframes*/           /*nombre que llevará la animación*/ 
@keyframes background-change {
  /*1er keyframe*/       /*nombre de la propiedad que se animará*/
  0%   { width: red; }
  /*2do keyframe*/
  50%  { background-color: yellow; }
  /*3er keyframe*/
  100% { background-color: red; }
}

Uso de keyframes

            
div {
  /*describe la animación a usar*/                /*definida en el ejemplo anterior*/
  animation-name: background-change;
  /*cuánto tardará en completar*/
  animation-duration: 0s;
  /*cuánto tardará en empezar*/
  animation-delay: 0s;
}
